#  Periwinkle Punk: The Cristina Ochoa Author Website

**Periwinkle Punk** is a modern, "quaint and refreshing" digital sanctuary for author Cristina Ochoa. This project is a high-performance Next.js application designed to bridge the gap between traditional storytelling and modern interactive technology.

---

##  Key Features

-   ** Dynamic Library:** A fully automated "Books" section using Next.js Dynamic Routing (`[slug]`) and TypeScript interfaces for seamless content management.

-   ** The Moodboard:** An interactive AI-powered chat interface designed to provide whimsical book recommendations based on user moods.

-   ** Seamless Dark Mode:** A custom theme provider that shifts the aesthetic from a "Creamy Morning" to a "Berry Night" experience.

-   ** Performance First:** Leveraging **React 19 Server Components** and **Next.js 16** for near-instant page transitions and industry-leading SEO.

-   ** Tailwind 4 Aesthetic:** A bespoke design system using the latest Tailwind CSS 4 `@theme` variables, featuring a periwinkle and berry-brown color palette.

---

## The Tech Stack

| Technology | Purpose |
| :--- | :--- |
| **Next.js 16** | The core framework for SSR, SSG, and File-System Routing. |
| **React 19** | Utilizing the latest React features and the **React Compiler** for automatic optimization. |
| **Tailwind CSS 4** | Utility-first styling with zero-config CSS variables. |
| **TypeScript 5** | Ensuring type safety. |
| **Lucide React** | A consistent, minimal icon set for a "quaint" UI. |
| **Vercel** | The deployment platform for seamless CI/CD and edge performance. |

##  Project Structure

```text
src/
├── app/             # Next.js App Router (Pages, Layouts, Loading, 404)
├── components/      # Reusable UI components (Navbar, Footer, Cards)
├── context/         # React Context for Global State (Theme Mode)
├── types/           # TypeScript interfaces for data modeling
└── globals.css      # Tailwind 4 configuration and global styles

```
